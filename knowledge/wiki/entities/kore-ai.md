# Kore.ai / ABL

The vendor (Kore.ai) and product layer (ABL — Artemis-ABL deterministic workflow platform)
whose value this knowledge base exists to substantiate. This is the **subject**, not an
external benchmark source; its claims need ABL production telemetry, not analyst reports.

## The mechanism that creates the value
**ABL's deterministic scripted workflows resolve routine work without an LLM call.** So its
marginal cost stays near an infrastructure floor (~$0.20 / interaction, default) while
pure-LLM rivals' cost per resolution climbs toward [$3+ by 2030](./gartner.md). That
divergence is the moat — and because value concentrates in the **low-complexity tier**, it is
durable. See [deterministic vs pure-LLM](../comparisons/deterministic-vs-pure-llm.md).

## Pricing alignment
ABL favors **consumption / outcome pricing** over per-seat pricing, which resolves the
[seat-decay contradiction](../concepts/seat-decay-contradiction.md): customer savings are
preserved as automation succeeds, rather than clawed back by seat-based fees.
